What is an AI Agent?

Unlike standard LLMs that wait for a prompt to provide a static response, an AI Agent is a system designed to interact with its environment to achieve a specific goal. It possesses autonomy, reactivity, and social ability.

Figure 1.1: The Perception-Action Loop in Autonomous Agents

Key Characteristics

  • check_circle Autonomy: Operating without direct intervention from humans or others.
  • check_circle Perception: Sensing the current state of the environment via data streams.
  • check_circle Memory: Maintaining a history of interactions to improve future decision making.
